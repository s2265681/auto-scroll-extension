# Privacy Policy — Auto Scroll Reader

**Last updated:** 2026-04-13

## Overview

Auto Scroll Reader is a Chrome browser extension that provides automatic page scrolling for hands-free reading. We are committed to protecting your privacy.

## Data Collection

**Auto Scroll Reader does not collect, store, transmit, or share any personal data or user information.**

Specifically, this extension:

- Does **not** collect personal identity information
- Does **not** collect browsing history or web activity
- Does **not** collect financial or payment information
- Does **not** collect health information
- Does **not** collect authentication credentials
- Does **not** collect communications data
- Does **not** collect location data
- Does **not** collect website content
- Does **not** use analytics, tracking, or telemetry of any kind
- Does **not** communicate with any external servers

## Local Storage

The extension uses Chrome's `storage.sync` API solely to save your personal preferences (scroll speed and on/off toggle state). This data:

- Is stored locally on your device and synced only through your own Chrome profile
- Is never sent to any third-party server
- Contains no personally identifiable information

## Permissions

| Permission | Purpose |
|-----------|---------|
| `activeTab` | Inject scrolling script into the current tab |
| `scripting` | Enable the auto-scroll content script on web pages |
| `storage` | Save your speed and toggle preferences locally |

## Third Parties

This extension does not use any third-party services, libraries, or SDKs. It is built entirely with vanilla JavaScript with zero external dependencies.

If this privacy policy is updated, changes will be reflected in this document with an updated date.
