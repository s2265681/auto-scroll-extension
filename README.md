# Auto Scroll Reader

> 一款轻量级 Chrome 自动滚动插件，让你解放双手，沉浸阅读。

## 功能特性

### 自动匀速滚动
- 开启后页面自动平滑向下滚动，适合阅读长文章、小说、技术文档、漫画等场景
- 智能识别页面滚动容器，兼容掘金、知乎、Medium 等各类网站

### 快捷键控制（无需打开插件面板）
| 按键 | 功能 |
|------|------|
| `Space` | 开始 / 暂停滚动 |
| `←` | 降低滚动速度 |
| `→` | 提高滚动速度 |

- 关闭插件面板后快捷键依然生效
- 到达最低/最高速度时页面内弹出提示
- 输入框内按键不会被拦截，不影响正常输入

### 速度调节
- 提供 1-10 档速度，通过滑动条或键盘左右键实时调节
- 调速时无需暂停，滚动过程中即时生效

### 智能暂停
- 用户滚动鼠标滚轮时自动暂停，按空格恢复
- 单击页面内容区域暂停（不影响链接和按钮点击）
- 触屏设备触摸滑动时自动暂停

### 总开关
- 顶部一键开关，关闭后插件所有功能停用，空格键恢复浏览器默认行为
- 开关状态自动保存，跨页面跨会话保持

## 安装

1. 下载或克隆本项目
2. 打开 Chrome，访问 `chrome://extensions/`
3. 开启右上角 **开发者模式**
4. 点击 **加载已解压的扩展程序**
5. 选择项目目录

## 使用

1. 打开任意网页
2. 按下 `Space` 即可开始自动滚动
3. 再按 `Space` 暂停，`←` `→` 调节速度
4. 也可以点击插件图标，通过面板控制

## 项目结构

```
auto-scroll-extension/
├── manifest.json   # 插件配置 (Manifest V3)
├── popup.html      # 弹出面板 UI
├── popup.js        # 面板交互逻辑
├── content.js      # 页面注入脚本（滚动核心）
└── icons/          # 插件图标
```

## 技术亮点

- **Manifest V3** — 遵循 Chrome 最新扩展标准
- **零依赖** — 纯原生 JS，无第三方库，体积极小
- **智能容器检测** — 自动识别 `overflow: auto/scroll/hidden` 的滚动容器，不局限于 `window.scrollBy`
- **主动注入** — 通过 `chrome.scripting` API 按需注入，无需刷新页面即可使用
- **状态持久化** — 基于 `chrome.storage.sync` 跨设备同步配置
